# sweet_cream
## https://tennk33.github.io/sweet_cream/   
**Sweet Cream은 배스킨라빈스 홈페이지를 참고하여 만든 아이스크림 브랜드 사이트입니다.**   
**카카오 맵 API와 카카오 이미지 검색 API를 연동하였으며 반응형으로 제작되었습니다.**

* **사용한 언어 :**
  * HTML5
  * CSS3
  * Javascript
  * jQuery
-------
## 메인화면 (PC버전)
<img src = "https://user-images.githubusercontent.com/84768543/128145656-de8d39c2-315a-4466-b33f-273044dbc41d.jpg" width="700px">   

**메인 화면의 이미지는 swiper 플러그인을 이용하여 자동으로 슬라이드 되도록 하였으며 드래그로 직접 슬라이드 할 수도 있습니다.**     

<img src = "https://user-images.githubusercontent.com/84768543/128149279-fe296289-f4fb-475c-9856-96ac5b0d8a50.jpg" width="700px">

**이벤트 부분 또한 드래그로 슬라이드 할 수 있으며 이미지 아래에 있는 동그라미를 눌러서 페이지를 넘길 수 있습니다.**   
**메인페이지의 레이아웃은 베스킨라빈스 홈페이지를 참고하여 만들었습니다.**

<img src = "https://user-images.githubusercontent.com/84768543/128149289-b0388213-505c-4d18-a2cd-bf8133d9e881.jpg" width="700px">   

**카카오 이미지 검색 API를 활용하여 다양한 아이스크림 사진들이 화면에 뜰 수 있게 하였으며 사이트 컨셉에 맞게 마우스를 올리면 이미지가 빨갛게 변하게 하였습니다.**
